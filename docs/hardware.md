# Recommended hardware

This project targets a Raspberry Pi with a USB audio interface.

## Suggested baseline
- Raspberry Pi 4/5 (2GB+ recommended)
- USB audio interface with stereo line-in
- Stable 5V power supply

## Notes
- Use short USB cables to avoid noise.
- Prefer interfaces with hardware gain if available.
- ALSA device names are usually `hw:1,0`, `hw:2,0`, etc.
